# Roulette Simulation

A simple yet thrilling web-based roulette simulation where users can experience the tension of pulling the trigger with a random bullet loaded.

**[Deployed at zijuny.dev/roulette.html](https://zijuny.dev/roulette.html)**

## Features

- **Reload**: Load the chamber with one bullet randomly positioned in a six-slot chamber.
- **Fire**: Pull the trigger to test your luck. A countdown and suspenseful ticking sound precede each shot.
- **Realistic Effects**: Experience a gunshot or misfire sound, accompanied by visual cues.

## Usage

1. **Open the webpage** to start.
2. Click **Reload** to reset the chamber with a single bullet.
3. Click **Fire** to initiate the countdown and pull the trigger.

## Tech Stack

- HTML, CSS, and JavaScript for structure, styling, and interactivity.
- Audio effects for added realism (ensure files are in the correct path).
